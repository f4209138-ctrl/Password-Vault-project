Secure Password Vault 🔐

A lightweight, locally-hosted desktop application that generates highly secure passwords and safely stores them using 3-tier AES encryption.

Download.exe View GitHub Repositories

Try the Generator

Experience the same 16-character secure generation algorithm used in the app.

🛡️ Triple-Factor Login

Requires exactly three master passwords to derive the decryption key, rendering brute-force attacks nearly impossible.

🔒 Local AES Encryption

Uses the cryptography.fernet library to encrypt your data locally. Your credentials never touch the cloud or leave your PC.

⚡ Standalone .exe

No Python setup required. Just download the executable and run it instantly on your Windows machine.
